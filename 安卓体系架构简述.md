 

- 应用层：Java语言 + API
  - Java语言，通过类，对API进行了封装。
  - 用户通过Java语言，操作API组件。
  - 或者説，Java面向对象编程，这里的对象就是API；所以説，安卓是Java的面向组件编程。
- 库层 + ART
  - ART就像安卓Java虚拟机，抽象了底层的细节，主要硬件层面的。但最终还是会调用本地的系统调用，系统调用，调用驱动完成对硬件的操作。
  - 库层，ART在运行时，会需要很多的库，就在这里。这些库，就是对接驱动程序的。
